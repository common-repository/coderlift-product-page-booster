Coderlift Product Page Booster plugin will add an extra field to add content after the cart in single product page

== Description ==
If Woocommerce shop owner wants to put some extra information or any type of data just after the cart to inhance the user experience which could be useful for increasing the sales, then this plugin is best.
This will add an extra field in product adding page in backend and the data will be showen in single product page just after the cart button.

== Installation ==
Upload the Coderlift Product Page Booster plugin to your Woocommerce site, Activate it, and enjoy.
